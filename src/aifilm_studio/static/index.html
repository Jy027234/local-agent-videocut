快速开始

                
    
                  
  1. 在左侧新建项目，填写项目名、一句话目标、类型和目标秒数。
    2. 
                  
  2. 进入“设置”，配置编剧策划、文生图、图生视频三个模型。
    3. 
                  
  3. 回到“策划验收”，生成策划草案，检查角色、场景和分镜，批准后同步分镜表。
    4. 
                  
  4. 进入“编排”，把角色/场景参考图设为全片或绑定到指定分镜，必要时调整分镜顺序。
    5. 
                  
  5. 进入“图片验收”，先上传角色/场景参考图，再生成关键帧图，逐张检查、返工或批准。
    6. 
                  
  6. 进入“视频验收”，基于已批准参考图生成视频，逐镜头检查、返工或批准。
    7. 
                  
  7. 进入“交付”，导出 edit pack，并发送到智能剪辑软件生成剪辑标准预览。
    8. 
                

              


              

                
配置三类模型

                

                  

                    编剧策划模型
                    
负责把需求 Brief 转成故事方向、角色卡、场景卡和分镜稿。通常使用文本或全模态模型。

                  

                  

                    文生图模型
                    
负责根据分镜、角色/场景参考图和提示词生成参考图、人物图或关键帧图。

                  

                  

                    图生视频模型
                    
负责基于已批准参考图和镜头运动提示词生成视频片段。

                  

                

                
每个模型卡都可以配置供应商模板、模型名、Base URL、Key 环境变量或 Key 文件路径。Key 文件路径适合本地试用，不会把密钥显示到页面结果里。

                
“测试此模型”和“测试模型链路”都会先弹出确认。真实文生图和图生视频可能产生费用，确认前请检查模型和 Key 是否正确。

              


              

                
生成编排

                
“编排”页用于管理参考图、分镜、关键帧和视频之间的生成关系。参考图可以设为全片，也可以拖到指定分镜；不方便拖拽时，也可以在参考图卡片中选择分镜并点击绑定。

                
分镜行可以拖动调整顺序。关键帧和视频列会显示当前状态，并提示未绑定参考图、需先批准关键帧等依赖缺口。

              


              

                
三阶段验收

                
FilmGen Studio 的核心不是一次性自动生成，而是让用户按顺序检查每一阶段产物。每个阶段都可以只处理未通过项、局部返工、保留历史版本，再批准进入下一阶段。

                
    
                  
  • 策划验收：检查故事、角色、场景和每条分镜，必要时只重写单个镜头。
    • 
                  
  • 图片验收：上传并锁定角色/场景参考图，检查关键帧是否符合角色、场景、构图和风格要求，必要时基于某个历史版本返工。
    • 
                  
  • 视频验收：左侧选择镜头，中间播放视频，右侧核对已批准关键帧、prompt、模型和历史版本。
    • 
                

              


              

                
产物与版本

                
策划稿、图片、视频和导出的 manifest 都会进入产物库。历史版本不会覆盖当前批准版本，可以设为左右对比、继续返工或删除。

                
删除生成物会同时移除登记；如果文件位于 FilmGen 工作目录内，也会清理对应本地文件。删除前请确认该版本不再需要进入交付。

              


              

                
交付剪辑

                
交付页导出的 edit pack 默认只包含已批准的策划、关键帧和视频。发送到智能剪辑软件后，会生成剪辑标准预览，用户确认后再进入剪辑执行链路。

                
如果 edit pack 里没有可用视频，可以在交付页手动填写输入视频路径，作为智能剪辑软件的素材来源。

              


              

                
发布前检查

                
    
                  
  • 确认三类模型都已配置模型名和 Key。
    • 
                  
  • 确认真实生成前会弹出用户确认。
    • 
                  
  • 确认已有项目可完成策划、图片、视频、交付全流程。
    • 
                  
  • 确认发布包里包含启动脚本、依赖说明、FilmGen Studio 和智能剪辑软件入口。
    • 
                  
  • 确认 `workspace` 下的运行产物不会作为源码提交，只作为本机数据保留。
    •
